Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • the !doctype declaration tells browser what document type it should be when opened
  • the title element defines the titles for a webpage. it's required and extremely important or Search Engine Optimization (SEO)
  • the style element defines the CSS styles associated with a website
  • the base element defines the base URL for a webpage
  • the link element connects an external resource to the HTML doc
  • the meta element defines metadata such as the character set, description, keywords, author, and viewport
  • the script element embeds client-side scripts in the HTML doc
  • the img element contains info about images displayed
  • the "anchor" creates links to the same webpage or other webpages
  • href attribute points to the url for the link
  • the alt attribute contains a text string that describes the appearance and functionality of the image

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • inline CSS is when you are styling an element directly within the HTML file
  • external CSS style sheet is preferred, and allows developers to keep all their CSS in a separate file, making design changes easier
  • the margin property indicates how much space we want around the outside of an element, it is undetected on the page but adds a buffer, if set to 0 we are indicating we want NO extra space around the element
  • padding property adds space around the content inside of an element
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained
  • text-align centers a title or heading
  • img assigns property to any image
  • the rel attribute specifies the relationship between the current document and the linked document
  • the href attribute specifies the location (URL) of the external resource, it's using a relative path. location is derived from the current location
  • .card the period preceding designates this as a class selector, any HTML element assigned to the class "card" eill gain these properties from the CSS
